# 快速開始指南

5 分鐘內啟動你的 GeminiGen AI 圖片工作室！

## 📋 步驟總覽

1. 安裝依賴
2. 啟動開發伺服器
3. 取得 GeminiGen Token
4. 開始生成圖片

---

## 🚀 步驟 1：安裝依賴

開啟終端機，進入專案目錄：

\`\`\`bash
cd image-gen-site
npm install
\`\`\`

等待安裝完成（約 1-2 分鐘）。

---

## 💻 步驟 2：啟動開發伺服器

\`\`\`bash
npm run dev
\`\`\`

看到以下訊息表示成功：

\`\`\`
✓ Ready in 2.3s
○ Local:   http://localhost:3000
\`\`\`

開啟瀏覽器訪問 [http://localhost:3000](http://localhost:3000)

---

## 🔐 步驟 3：取得 GeminiGen Token

### 方法 A：使用自動腳本（推薦）

1. 開啟新分頁前往 [geminigen.ai](https://geminigen.ai)
2. 用 Google 帳號登入
3. 按 `F12` 開啟開發者工具
4. 點擊「Console」分頁
5. 在你的網站上複製 JavaScript 代碼
6. 貼到 Console 並按 Enter
7. 看到「✅ Token 已成功同步到星光工坊！」

### 方法 B：手動取得

1. 在 geminigen.ai 登入後，按 `F12`
2. 點擊「Application」分頁
3. 左側選擇「Local Storage」→「https://geminigen.ai」
4. 找到並複製：
   - `access_token`
   - `refresh_token`
5. 在網站的認證彈窗中手動輸入

---

## 🎨 步驟 4：開始生成圖片

1. 在輸入框中輸入提示詞，例如：
   - "一隻可愛的貓咪在花園裡玩耍"
   - "未來城市的夜景，霓虹燈閃爍"
   - "夢幻般的森林，陽光透過樹葉"

2. 點擊「生成圖片」按鈕

3. 等待 10-30 秒（視 API 速度而定）

4. 圖片生成完成！你可以：
   - 點擊圖片查看大圖
   - 點擊下載按鈕儲存到本地
   - 繼續生成更多圖片

---

## 🎯 提示詞技巧

### 基礎結構

\`\`\`
[主體] + [動作/狀態] + [環境/背景] + [風格/氛圍]
\`\`\`

### 範例

✅ **好的提示詞：**
- "一位穿著紅色洋裝的女孩在櫻花樹下跳舞，夕陽西下，溫暖的光線，動漫風格"
- "賽博龐克城市街道，霓虹燈招牌，雨後濕潤的地面，未來感，高細節"

❌ **不好的提示詞：**
- "貓" （太簡單）
- "一隻貓一隻狗一個人在公園裡吃飯看書玩遊戲" （太複雜）

### 常用關鍵字

- **風格**：寫實、動漫、水彩、油畫、賽博龐克、蒸汽龐克
- **光線**：夕陽、晨光、月光、霓虹燈、柔和光線
- **氛圍**：夢幻、神秘、溫暖、冷酷、寧靜
- **品質**：高細節、4K、專業攝影、電影感

## 🛠️ 常見問題

### Q: 生成失敗怎麼辦？

**A:** 檢查以下項目：
1. Token 是否正確且未過期
2. 網路連接是否正常
3. 提示詞是否包含敏感內容
4. 查看瀏覽器控制台的錯誤訊息

### Q: 生成速度很慢？

**A:** 這是正常的，AI 圖片生成通常需要 10-30 秒。影響因素：
- API 伺服器負載
- 圖片複雜度
- 網路速度

### Q: 可以生成多大的圖片？

**A:** 取決於 GeminiGen API 的限制，通常為 512x512 或 1024x1024。

### Q: Token 會過期嗎？

**A:** 會的。如果遇到認證錯誤，請重新取得 Token。

## 🎓 進階功能

### 批次生成

想要生成多張圖片？只需：
1. 生成第一張
2. 修改提示詞
3. 再次生成
4. 所有圖片都會保留在畫廊中

### 下載圖片

- 點擊圖片上的下載按鈕
- 或右鍵點擊圖片 → 另存圖片

### 清除歷史

重新整理頁面即可清除所有生成的圖片。
